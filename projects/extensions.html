Here you'll find a list of all of the GNOME 3 shell extensions that I use. I feel that with the third major version of GNOME, a lot of fluff and unneeded content has been brought forth. It feels like I should be using it on my phone or a touch screen device or something rather than an actual computer. From having to drag up the lock screen to unlock it to random changes like scheduling, a social menu, the "hot" upper left and lower right corners, and the removal of a window list panel, it's gotten a lot worse in my opinion. I use a pretty hefty list of extensions to keep GNOME running how I want without having to go back to something like GNOME 2, GNOME Classic, or MATE. If you like a user experience with no fluff, quick response times, and usability over looks, these extensions are for you.

This is actually my laptop setup, my desktop is a lot different. Laptop is a ThinkPad T60, and these extensions ran fine with only a gigabyte of RAM. I'm on 4GB of RAM now, but just know that this will run on a system with even the lowest amount of resources.
